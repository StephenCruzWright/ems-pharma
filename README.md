# EMS Pharma Website Starter

Modern, production-ready pharmaceutical company website starter built with **Next.js**, **TypeScript**, **Tailwind CSS**, and **Vitest**.

## Features

- Industry-standard pharma landing architecture (hero, therapeutic areas, capabilities, quality, FAQ, contact)
- Responsive design and accessible semantic structure
- Unit tests for core rendering and interactive FAQ behavior
- CI workflow for linting, type-checking, tests, and production build
- Ready for deployment to **Vercel**

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Quality Checks

```bash
npm run check
npm run build
```

## Deployment

Deploy on Vercel with default Next.js settings.
